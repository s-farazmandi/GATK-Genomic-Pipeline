**Project Title:** GATK Genomic Pipeline 

**Description:** <br />
This project utilizes the GATK toolkit for genomic variant analysis, focusing on SNP and Indel identification from DNA sequencing data of a trio. The pipeline includes pre-processing, variant calling, filtering, and annotation stages, leveraging tools like FastQC, Picard, and ANNOVAR. <br />
This project was carried out for academic purposes as a final project for the Computational Genomics MSc course at Tarbiat Modares University, Tehran, Iran, July 2022.
